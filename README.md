## Проект MESTO

https://notabene91.github.io/MESTO/

Учебный проект Яндекс.Практикума с импользованием
нативного JS и Webpack.

## Версия:  
0.0.2

## Инструкция по установке:

### -установить npm
```
npm install
```
### -запустить проект в режиме разработки
```
npm run dev
```
### -собрать проект
```
npm run build
```
### -добавить проект на GitHub Pages
```
npm run deploy
```
